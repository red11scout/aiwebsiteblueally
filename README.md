# BlueAlly AI Landing Page

A modern, interactive landing page for BlueAlly AI services built with React, TypeScript, and Tailwind CSS.

## Features

- **8-Part Framework**: BlueAlly's holistic AI transformation methodology
- **7-Step Use Case Methodology**: Systematic approach to AI value creation
- **Interactive Elements**: Expandable sections, animated charts, and data visualizations
- **Day/Night Mode**: Full theme switching support
- **Responsive Design**: Mobile, tablet, and desktop optimized
- **Hemingway-style Copy**: Clear, direct, professional prose

## Tech Stack

- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Recharts
- shadcn/ui components

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

## Project Structure

```
client/
  src/
    components/
      sections/     # Page sections (Hero, Framework, Process, etc.)
      ui/           # shadcn/ui components
    pages/          # Page components
    contexts/       # React contexts (Theme)
    hooks/          # Custom hooks
    lib/            # Utilities
server/             # Express server (for production)
shared/             # Shared types and constants
```
